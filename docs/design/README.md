# Design documents

Internal design documents and proposals for `open_pg_tde`. These describe
planned or proposed work and are distinct from the user documentation under
`documentation/`, which describes shipped features.

Each document carries a status (Proposed, Accepted, Implemented, or Rejected) and
a date.
