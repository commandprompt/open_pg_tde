# Fortanix KMIP server configuration

`open_pg_tde` is compatible with Fortanix Data Security Manager (DSM) via the KMIP protocol. For a full setup guide, see [the Fortanix KMIP documentation here](https://support.fortanix.com/docs).

For more information on adding or modifying the provider, see the [KMIP provider functions](../functions.md) topic.
